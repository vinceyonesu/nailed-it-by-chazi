# Nailed It by Chazi

A mobile-responsive booking web app for a home-based nail salon, built with **Next.js 14**, **Tailwind CSS**, and **Firebase v10**.

## Features

- 4-step booking flow: Service → Schedule → Contact → GCash Payment
- Firebase Firestore for booking storage
- Firebase Storage for GCash receipt uploads
- Mobile-first design with soft pinks, nudes, and rose gold
- Booking policy enforcement (₱50 non-refundable slot fee)

## Getting Started

1. Copy `.env.local.example` to `.env.local` and fill in your Firebase config values.

2. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
